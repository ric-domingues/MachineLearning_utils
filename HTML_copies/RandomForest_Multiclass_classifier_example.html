Random Forest Multiclass Classifier example¶

In this notebook, a Random Forest Classification Model employing the iris dataset, which consists of a dataset with features (e.g. petal width, sepal lenght, etc) and species for flowers of the iris genus). Additional information about this dataset can be found here.
